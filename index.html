📋 Airtable Setup Guide

Buka file HTML ini di text editor, cari bagian CONFIG di atas, isi token dan Base ID.

1. Get API Token

Buka airtable.com/create/tokens
→ Create new token
→ Scope: data.records:read + data.records:write
→ Access: pilih Base kamu

2. Get Base ID

Buka Airtable Base kamu
→ URL: airtable.com/appXXXXX/...
→ Bagian appXXXXX = Base ID

3. Buat 3 Tabel di Airtable
Sales Activity

• Tanggal (Date)
• Client (Text)
• Sales Name (Text)
• Activity Type (Select)
• MoM (Long text)
• Potential (Select)
• Next Step (Long text)
• Tanggal FU (Date)
• Status FU (Select)
• Project ID (Text)

Project Tracking

• Client (Text)
• Sales Name (Text)
• Total Value (Number)
• Notes (Long text)
• Contract Date (Date)
• Handover Date (Date)
• Terms JSON (Long text)

Members

• Name (Text)
• Role (Single select)
• Note (Text)
• Password (Text)

✅ Select field values to add:
Activity Type: Call, Email, Meeting
Potential: Hot, Warm, Cold, Win
Status FU: Pending, Done, Win
Role (Members): sales, head
Ganti Password (Admin / Head)
Backup Data

Export snapshot dari cache sebagai JSON. Data utama tersimpan di Airtable.
